# 🏦 NTC Bank — Simulador de Banco Digital

Interface completa de um banco digital moderno desenvolvida com HTML, CSS e JavaScript puro.

## 🔗 Demo ao vivo
**[ntcbank.netlify.app](https://ntcbank.netlify.app)**

## 📱 Telas
| Tela | Descrição |
|------|-----------|
| **Login** | Autenticação com card glassmorphism animado |
| **Cadastro** | Formulário em 3 steps com validação e máscara |
| **Dashboard** | Saldo, ações rápidas, gráfico semanal e transações |
| **Extrato** | Histórico de movimentos com resumo receita/despesa |
| **Cartão** | Wallet card, spending, cashback e modal add card |
| **Pagamento** | Teclado numérico, contatos, confirmação e sucesso |
| **Perfil** | Configurações, segurança, preferências e dark mode |

## ✨ Funcionalidades
- 🌙 Dark/Light mode persistido via localStorage
- 💳 Glassmorphism em cards, navbar e modais
- 🎨 Orbs de luz animados no fundo
- 👁️ Toggle de saldo visível/oculto
- ⌨️ Teclado numérico funcional na tela de pagamento
- ✅ Modal de confirmação de pagamento com tela de sucesso
- 📋 Formulário de cadastro em 3 etapas com máscaras
- 🔗 Navegação completa entre todas as telas

## 🛠️ Tecnologias
- HTML5
- CSS3 (Glassmorphism, variáveis CSS, animações)
- JavaScript Vanilla
- Google Fonts — Sora + DM Sans + DM Mono
- Deploy: Netlify | Versionamento: Git + GitHub
